Odomate – User Manual

Introduction

Odomate is an application for vehicle management, navigation, and trip tracking with AI-powered maintenance recommendations.

1. Getting Started

  1. Install and open the application.
  2. Create an account or log in (email).
  3. Add your first vehicle (make, model, year, VIN optional).
  4. Set units of measurement (km/mi) and language in settings.

2. Vehicles

  • Add/edit/delete vehicles.
  • Fields: make, model, year, VIN, license plate (optional).
  • Update odometer; attachments (documents/notes).
  • Multiple vehicles supported.

3. Trips

3.1 Manual Logging (Free)

  • Create trip: start/finish, distance, notes.
  • Categories and tags.

3.2 Navigation and Automatic Tracking (Premium)

  1. Find destination (address/place/coordinates).
  2. Add waypoints (optional).
  3. Start navigation and follow instructions.
  4. Automatic tracking works during active navigation: records distance and route in background.
  5. Automatic rerouting when deviating from route.

Notes:
- Maps: Mapbox.
- Navigation and routing: Valhalla (self-hosted), Mapbox as fallback.
- Search: Nominatim.
- Battery impact: GPS consumes more power.
- Always follow traffic laws.

4. Maintenance

4.1 AI Maintenance Plans

  • Based on vehicle data and mileage.
  • Recommendations for service intervals and components.

4.2 Reminders

  • Reminders are sent automatically based on AI maintenance plans and set intervals.
  • Enable push notifications in settings (device permissions required).

5. Synchronization

  • Account via Supabase; data syncs across devices.
  • Offline-first: changes sent when connection restored.
  • Conflicts: last edit wins; manual correction if needed.

6. Settings

  • Units (km/mi), language, time format (12/24 hour), temperature format (°C/°F).
  • Notifications (maintenance, tracking).
  • Privacy: account deletion, analytics opt-out.
  • Map options: traffic, 3D, zoom, avoid tolls, avoid highways, avoid ferries.

7. Accuracy Tips

  • Keep GPS enabled; adjust power saving if necessary.
  • Movement-bearing technology is always used for determining direction of travel.
  • Keep the app updated.

8. Troubleshooting

  • No GPS position: check permissions and location services.
  • Navigation stopped: check internet/GPS, restart app.
  • Sync stuck: check connection and retry later.
  • Contact support (below).

9. Privacy

  • Location used only during active navigation/tracking.
  • Anonymous analytics for quality improvement.
