Odomate – Guide Utilisateur

Introduction

Odomate est une application de gestion de véhicule, de navigation et de suivi des trajets avec recommandations d'entretien basées sur l'IA.

1. Bien démarrer

  1. Installez et ouvrez l'app.
  2. Créez un compte ou connectez‑vous (e‑mail).
  3. Ajoutez votre premier véhicule (marque, modèle, année, VIN optionnel).
  4. Réglez unités (km/mi) et langue dans les paramètres.

2. Véhicules

  • Ajouter/modifier/supprimer un véhicule.
  • Champs : marque, modèle, année, VIN, immatriculation (optionnel).
  • Mettre à jour le kilométrage ; joindre documents/notes.
  • Multi‑véhicules pris en charge.

3. Trajets

3.1 Saisie manuelle (Gratuit)

  • Créer un trajet : début/fin, distance, notes.
  • Ajouter catégories et tags.

3.2 Navigation et suivi automatique (Premium)

  1. Recherchez une destination (adresse/lieu/coordonnées).
  2. Ajoutez des étapes (optionnel).
  3. Lancez la navigation et suivez les instructions.
  4. Le suivi automatique fonctionne pendant la navigation active : enregistre distance et itinéraire en arrière‑plan.
  5. Recalcul automatique en cas de sortie d'itinéraire.

Notes :
- Cartes : Mapbox.
- Navigation et itinéraires : Valhalla (auto-hébergé), Mapbox comme solution de secours.
- Recherche : Nominatim.
- Batterie : le GPS consomme davantage.
- Respectez toujours le code de la route.

4. Entretien

4.1 Plan IA

  • Basé sur les données véhicule et le kilométrage.
  • Propose des échéances d'entretien et pièces à prévoir.

4.2 Rappels

  • Les rappels sont envoyés automatiquement sur la base des plans d'entretien IA et des intervalles définis.
  • Activez les notifications push dans les paramètres (autorisation d'appareil requise).

5. Synchronisation

  • Compte via Supabase ; données synchronisées entre appareils.
  • Offline‑first : les modifications sont envoyées à la reconnexion.
  • Conflits : dernière modification prioritaire ; corriger manuellement si besoin.

6. Paramètres

  • Unités (km/mi), langue, format de l'heure (12/24 heures), format de température (°C/°F).
  • Notifications (entretien, suivi).
  • Confidentialité : suppression du compte, opt‑out d'analyses.
  • Options de carte : trafic, 3D, échelle, éviter les péages, éviter les autoroutes, éviter les ferries.

7. Conseils de précision

  • Laissez le GPS activé ; ajustez l'économie d'énergie si nécessaire.
  • La technologie movement‑bearing est toujours utilisée pour déterminer la direction du mouvement.
  • Mettez l'app à jour.

8. Dépannage

  • Pas de position : vérifiez autorisations et services de localisation.
  • Navigation arrêtée : vérifiez Internet/GPS, redémarrez l'app.
  • Sync bloquée : vérifiez la connexion et réessayez.
  • Contactez le support (ci‑dessous).

9. Confidentialité

  • Localisation uniquement en navigation/suivi actifs.
  • Analyses anonymes pour améliorer l'app.
  • Détails : legal/PRIVACY_POLICY_FR.md, legal/GDPR_COMPLIANCE_FR.md.
