Вступ

Odomate — додаток для керування автомобілем, навігації та обліку поїздок з ШІ‑рекомендаціями з обслуговування.

1. Початок роботи

  1. Встановіть та відкрийте додаток.
  2. Створіть акаунт або увійдіть (e‑mail).
  3. Додайте перший автомобіль (марка, модель, рік, VIN опціонально).
  4. Задайте одиниці вимірювання (км/ми) та мову в налаштуваннях.

2. Автомобілі

  • Додавання/редагування/видалення авто.
  • Поля: марка, модель, рік, VIN, номер (опціонально).
  • Оновлення пробігу; вкладення (документи/замітки).
  • Підтримка кількох авто.

3. Поїздки

3.1 Ручний облік (Безкоштовно)

  • Створення поїздки: старт/фініш, дистанція, замітки.
  • Категорії та теги.

3.2 Навігація та автоматичне відстеження (Преміум)

  1. Знайдіть ціль (адреса/місце/координати).
  2. Додайте проміжні точки (за бажанням).
  3. Запустіть навігацію та слідуйте інструкціям.
  4. Автоматичне відстеження працює під час активної навігації: записує дистанцію та маршрут у фоні.
  5. Автоперебудова маршруту при відхиленні.

Примітки:
- Карти: Mapbox.
- Навігація та маршрути: Valhalla (власний сервер), Mapbox як резервний.
- Пошук: Nominatim.
- Вплив на батарею: GPS споживає більше енергії.
- Завжди дотримуйтесь ПДР.

4. Обслуговування

4.1 ШІ‑плани ТО

  • На основі даних авто та пробігу.
  • Пропозиції щодо інтервалів та витратних матеріалів.

4.2 Нагадування

  • Нагадування приходять автоматично на основі ШІ‑планів обслуговування та встановлених інтервалів.
  • Увімкніть push‑сповіщення в налаштуваннях (потрібні дозволи пристрою).

5. Синхронізація

  • Акаунт через Supabase; дані синхронізуються між пристроями.
  • Offline‑first: зміни відправляються при відновленні мережі.
  • Конфлікти: перемагає остання правка; при необхідності виправте вручну.

6. Налаштування

  • Одиниці (км/ми), мова, формат часу (12/24 години), формат температури (°C/°F).
  • Сповіщення (ТО, трекінг).
  • Конфіденційність: видалення акаунту, відмова від аналітики.
  • Опції карти: трафік, 3D, масштаб, уникати платних доріг, уникати автомагістралей, уникати водних переправ.

7. Поради щодо точності

  • Тримайте GPS увімкненим; при необхідності налаштуйте енергозбереження.
  • Для визначення напрямку руху завжди використовується технологія «movement‑bearing».
  • Оновлюйте додаток.

8. Вирішення проблем

  • Немає GPS‑позиції: перевірте дозволи та служби геолокації.
  • Навігація зупинилася: перевірте інтернет/GPS, перезапустіть додаток.
  • Синхронізація зависла: перевірте підключення та повторіть пізніше.
  • Зв'яжіться з підтримкою (нижче).

9. Конфіденційність

  • Локація використовується тільки при активній навігації/трекінгу.
